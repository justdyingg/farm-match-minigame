## 🎮 核心玩法

- **🏡 农场经营**  
  解锁土地 → 播种 → 浇水 → 生长 → 收获，体验完整的农场循环。
- **💎 三消关卡**  
  8×8 棋盘，交换相邻方块完成三消，获得种子奖励。
- **🌤️ 天气系统**  
  基于浏览器定位获取实时天气；定位失败 / 超时自动使用默认城市。
- **🎒 种子背包**  
  背包支持播种与查看，数据本地持久化，刷新不丢失。

## 🏗️ 项目架构

assets/scripts/
├── core/ ← 基础设施（单例、事件总线、音频、存储）
├── api/ ← 外部接口（天气 API 封装）
├── game/ ← 核心玩法（农场管理、三消逻辑）
├── ui/ ← UI 组件（弹窗、数字标签、加载动画）
├── data/ ← 数据模型（土地、方块、存档结构）
├── vfx/ ← 特效（浇水动画、金币飞出）
└── plugin/ ← 屏幕适配、背景拉伸

---

### 设计模式与架构亮点

- **单例模式**：自定义 `Singleton` 基类管理全局唯一服务（音频、存储、事件）
- **发布-订阅模式**：`EventManager` 实现模块解耦，方块点击、游戏状态等通过事件传递
- **有限状态机**：土地生命周期（`LOCKED → EMPTY → SEEDLING → BUD → BLOOMING`）
- **MVC 分层**：严格遵循 `数据 (data)` - `逻辑 (game)` - `表现 (ui)` 分离
- **对象池**：金币特效节点复用，减少 GC 开销
- **屏幕适配**：动态判断设备宽高比，自动切换 `FIXED_HEIGHT` / `FIXED_WIDTH`
